# 老年人吞咽障碍与软食指南 (Soft Diet for Dysphagia)

## 1. 疾病概述
吞咽障碍（Dysphagia）是老年人常见的问题，特别是伴有中风后遗症、帕金森病、食管疾病或牙齿缺失的老人。吞咽困难可能导致营养不良、脱水，甚至误吸性肺炎（老年人死亡主要原因之一）。

## 2. 核心饮食原则 (安全第一)
- **改变食物质地**：
  - **泥状**（Puree）：像婴儿米糊一样，无颗粒，只需吞咽无需咀嚼。
  - **剁碎**（Minced）：切成小米粒大小，易成团，容易吞咽。
  - **软食**（Soft）：蒸煮至软烂，可用舌头压碎，如鱼肉、豆腐。
- **调整液体稠度**：
  - **稀流质**（水、清汤）：最容易引起呛咳误吸！
  - **浓流质**（酸奶、芝麻糊）：相对安全。
  - **增稠剂**：在水中加入专门的凝固粉，调成“花蜜状”或“布丁状”，降低流速，增加安全性。
- **一口量**：每口5-10ml，吃完一口再吃下一口。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（安全推荐）
- **软烂主食**：稠粥、软面条、蒸红薯泥、土豆泥、馒头泡汤。
- **优质蛋白**：肉末蒸蛋、豆腐脑、鱼泥、虾滑、肉丸子（无筋）。
- **蔬菜水果**：蒸熟的南瓜泥、胡萝卜泥、香蕉泥、煮烂的冬瓜。
- **增稠饮品**：加入增稠剂的水、果汁、牛奶、藕粉。

### ❌ 红灯食物（极度危险！）
- **干硬食物**：坚果、炒豆子、硬饼干、锅巴。
- **粘性食物**：汤圆、年糕、粽子、糯米糍（极易卡喉窒息）。
- **多渣食物**：芹菜、韭菜、豆芽、未去皮的番茄（纤维难以咬断）。
- **混合质地**：如清汤面（汤水流速快，面条慢，易呛咳）、带皮带籽的水果（如西瓜）。

## 4. 推荐食谱示例 (针对轻度吞咽困难)
- **早餐**：牛奶泡蛋糕（软烂） + 香蕉泥。
- **午餐**：肉末豆腐羹（勾芡增稠） + 蒸蛋羹 + 软米饭（稍加水煮软）。
- **加餐**：芝麻糊或藕粉。
- **晚餐**：冬瓜肉丸汤（肉丸剁细） + 龙须面（煮烂）。

## 5. 特别提示
- **进食姿势**：坐直身体（90度），头稍前倾（低头吞咽，保护气道）。
- **口腔清洁**：餐后务必漱口或刷牙，防止食物残渣误入气管引发肺炎。
- **避免分心**：吃饭时不要说话、看电视，专心吞咽。
