# 慢病专病饮食指南：高血压 (Hypertension)

## 1. 核心原则 (DASH 饮食法)

### **DASH (Dietary Approaches to Stop Hypertension) 饮食要点**
1. **高钾、高镁、高钙、高膳食纤维**
   - 多吃新鲜蔬菜（尤其是绿叶菜）、水果、低脂奶制品。
   - 钾：每日建议3500-4700mg（肾功能不全者慎用）。
   - 镁：每日建议300-420mg（如坚果、豆类）。
   - 钙：每日建议1000-1200mg（奶类、豆制品）。

2. **限制饱和脂肪、胆固醇**
   - 减少红肉（猪牛羊肉）、肥肉、动物内脏、全脂奶。
   - 增加白肉（鱼、去皮禽肉）、植物油（橄榄油、菜籽油）。

3. **限制精制糖**
   - 少吃甜点、含糖饮料、果汁。

## 2. 每日食盐限制

### **严格限盐 (< 5g/天)**
- **标准**：每日食盐总量不超过5克（约一啤酒瓶盖）。
- **隐形盐**：警惕酱油、味精、咸菜、腊肉、火腿、方便面、腐乳等高钠食品。
- **替代调味**：使用葱、姜、蒜、醋、柠檬汁、香草等天然香料提味。

## 3. 食物选择红绿灯

### **绿灯食物 (推荐)**
- **蔬菜**：菠菜、油菜、芹菜、空心菜、茄子、西红柿、黄瓜、苦瓜。
- **水果**：香蕉（高钾）、苹果、橘子、猕猴桃、西瓜。
- **全谷物**：燕麦、糙米、荞麦、全麦面包。
- **蛋白质**：低脂牛奶、酸奶、豆浆、豆腐、鱼虾、去皮鸡肉。
- **坚果**：核桃、杏仁、腰果（原味，无盐）。

### **黄灯食物 (限量)**
- **红肉**：瘦猪肉、牛肉、羊肉（每日<50g）。
- **鸡蛋**：每日1个（胆固醇高者隔日吃黄）。
- **油脂**：植物油（每日<25g）。

### **红灯食物 (避免)**
- **腌制食品**：咸菜、榨菜、泡菜、咸蛋、皮蛋。
- **加工肉类**：香肠、腊肉、火腿、午餐肉。
- **高钠零食**：薯片、话梅、方便面调料包。
- **酒精**：烈性酒、啤酒（影响降压药效果）。

## 4. 生活方式干预

1. **减重**：BMI控制在24以下，腹围男性<90cm，女性<85cm。
2. **运动**：每周至少150分钟中等强度有氧运动（快走、慢跑、游泳）。
3. **戒烟限酒**。
4. **规律作息**，避免熬夜。
5. **监测血压**，早晚各一次。
