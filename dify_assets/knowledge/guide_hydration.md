# 老年人补水与水化指南 (Hydration)

## 1. 概述
随着年龄增长，老年人的**口渴中枢**功能退化，感觉不到口渴，加上肾脏浓缩尿液能力下降、使用利尿剂等因素，导致老年人极易发生**脱水**（Dehydration）。脱水会引发便秘、结石、感染，甚至增加心脑血管意外风险。

## 2. 核心原则
- **主动喝水**：不等口渴才喝水，养成定时喝水习惯。
- **少量多次**：每次100-200ml，全天分散进行。
- **推荐量**：
  - 一般老年人：每日1500-1700ml（约7-8杯）。
  - 心肾功能不全者：**严格遵医嘱限水**（可能需<1000ml）。
  - 发热、腹泻、出汗多：适当增加。

## 3. 推荐饮品（红灯/绿灯）

### ✅ 绿灯饮品（首选）
- **白开水**：最安全、最经济。
- **淡茶水**：绿茶、红茶、花茶（提神、抗氧化）。
- **豆浆/牛奶**：既补水又补营养。
- **汤羹**：蔬菜汤、鸡蛋汤、绿豆汤（少盐少油）。
- **新鲜果汁**：适量（最好带果肉，避免高糖）。

### ❌ 红灯饮品（限制/避免）
- **浓茶、浓咖啡**：利尿作用强，反而导致水分流失。
- **含糖饮料**：可乐、果汁饮料（增加血糖、龋齿风险）。
- **酒精**：严重利尿，加重脱水。
- **冰水**：刺激胃肠道，引起痉挛。

## 4. 补水时刻表（示例）
- **晨起（7:00）**：温开水一杯（200ml），润肠通便，稀释血液。
- **上午（10:00）**：淡茶水/柠檬水一杯。
- **午餐前（11:30）**：清汤一碗（润滑食道）。
- **下午（15:00）**：绿豆汤/牛奶一杯。
- **晚餐前（17:30）**：温开水一杯。
- **睡前（20:00）**：少量温水（50-100ml，避免夜尿频繁）。

## 5. 观察指标
- **尿液颜色**：
  - 淡黄色/清亮：水分充足。
  - 深黄色/茶色：严重缺水，需立即补充！
- **皮肤弹性**：捏起手背皮肤，松开后迅速恢复平整说明水分正常；若恢复缓慢可能脱水。
- **口唇干燥**、**眼窝凹陷**：脱水信号。
