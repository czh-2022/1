# 慢病专病饮食指南：痛风 (Gout)

## 1. 核心原则 (低嘌呤)

### **痛风/高尿酸血症饮食**
1. **限制高嘌呤食物**
   - 每日嘌呤摄入量控制在300mg以内。
   - 避免急性发作。

2. **限制高果糖摄入**
   - 果糖会竞争性抑制尿酸排泄，升高血尿酸。
   - 避免甜味水果、含糖饮料。

3. **适量蛋白质**
   - 每日1.0-1.2g/kg。
   - 植物性蛋白质（如豆腐、豆浆）对血尿酸影响较小。

4. **充足水分**
   - 每日饮水量2000-3000ml。
   - 促进尿酸排泄，预防肾结石。

5. **戒烟限酒**
   - 酒精（尤其是啤酒、白酒）干扰尿酸排泄。

## 2. 嘌呤含量红绿灯

### **低嘌呤食物 (可食用)**
- **主食**：精米、白面、馒头、面条、苏打饼干。
- **蔬菜**：绝大多数蔬菜（如白菜、萝卜、黄瓜、西红柿、土豆）。
- **水果**：大多数水果（如苹果、梨、桃、西瓜、葡萄）。
- **奶类**：牛奶、酸奶、奶酪。
- **蛋类**：鸡蛋、鸭蛋、鹌鹑蛋。

### **中嘌呤食物 (限量)**
- **肉类**：猪肉、牛肉、羊肉、鸡肉、鸭肉（去皮）。
- **鱼类**：草鱼、鲤鱼、鲫鱼、鲈鱼。
- **豆类**：干豆（如黄豆、黑豆、红豆）、豆腐干。
- **蔬菜**：芦笋、菠菜（焯水去草酸）、蘑菇（鲜）、菜花。

### **高嘌呤食物 (避免)**
- **内脏**：肝、肾、脑、肠、心。
- **海鲜**：贝类（蛤蜊、扇贝）、虾蟹、沙丁鱼、凤尾鱼。
- **肉汤**：浓肉汤、火锅汤、鸡精。
- **酒类**：啤酒、白酒。
- **饮料**：可乐、雪碧、果汁、蜂蜜水。

## 3. 特殊注意事项

1. **急性发作期**
   - 严格禁食高嘌呤食物。
   - 主食以细粮为主（大米粥、面条）。
   - 多喝水，每日3000ml以上。
   - 避免剧烈运动。

2. **缓解期**
   - 适量摄入中嘌呤食物（肉类煮后弃汤食用）。
   - 保持理想体重，避免肥胖。

3. **降尿酸食物推荐**
   - **樱桃**：含有花青素，有助于降低尿酸。
   - **芹菜**：利尿，促进尿酸排泄。
   - **黄瓜**：低嘌呤，利尿。
   - **苏打水**：碱化尿液，预防肾结石。
