# 老年人营养食谱集 (Recommended Recipes)

## 1. 早餐篇 (Breakfast)

### 方案 A：杂粮粥 + 鸡蛋
- **主食**：杂粮粥（小米、燕麦、红豆）1碗（约250g）。
- **蛋白质**：煮鸡蛋1个。
- **蔬菜**：凉拌黄瓜（少油少盐）或焯水菠菜1小碟。
- **点评**：富含膳食纤维，升糖指数低，适合大多数老年人。

### 方案 B：牛奶燕麦 + 全麦面包
- **主食**：牛奶燕麦片（纯燕麦，无糖）1碗。
- **蛋白质**：低脂奶250ml。
- **搭配**：全麦面包1片，可抹少许芝麻酱。
- **点评**：补钙首选，制作简单，易于消化。

### 方案 C：豆浆 + 菜包
- **主食**：素菜包子1个（约100g）。
- **蛋白质**：无糖豆浆1杯（约300ml）。
- **加餐**：坚果碎（核桃/杏仁）10g。
- **点评**：植物蛋白丰富，适合乳糖不耐受者。

## 2. 午餐篇 (Lunch)

### 方案 A：清蒸鱼 + 杂粮饭
- **主菜**：清蒸鲈鱼（约100g），姜葱去腥，少盐。
- **配菜**：蒜蓉西兰花（焯水后淋少许橄榄油）。
- **主食**：二米饭（大米+小米）1碗（约150g）。
- **汤**：紫菜蛋花汤（清淡）。

### 方案 B：番茄炖牛腩 + 荞麦面
- **主菜**：番茄炖牛腩（牛腩去油，切小块炖烂），少盐。
- **配菜**：凉拌木耳洋葱（少醋）。
- **主食**：荞麦面条1碗。

### 方案 C：虾仁豆腐 + 馒头
- **主菜**：虾仁烩豆腐（嫩豆腐，虾仁去线）。
- **配菜**：清炒油麦菜。
- **主食**：全麦馒头1个。

## 3. 晚餐篇 (Dinner)

### 方案 A：冬瓜肉丸汤 + 花卷
- **主菜**：冬瓜瘦肉丸汤（肉丸用鸡胸肉或瘦猪肉泥制作，淀粉少许）。
- **配菜**：凉拌海带丝。
- **主食**：小花卷1个。

### 方案 B：香菇滑鸡粥
- **主食**：香菇鸡肉粥（大米熬煮至软烂，加入鸡肉丝和香菇丁）。
- **配菜**：白灼菜心。
- **点评**：易消化，适合牙口不好的老人。

### 方案 C：素炒三丝 + 杂粮饭
- **主菜**：素炒三丝（胡萝卜丝、土豆丝、青椒丝）。
- **配菜**：清蒸南瓜（替代部分主食）。
- **主食**：黑米饭半碗。

## 4. 加餐建议 (Snacks)

- **上午加餐 (10:00)**：低糖水果（如苹果半个、柚子2瓣、草莓5颗）。
- **下午加餐 (15:00)**：原味坚果（核桃2个或杏仁10颗）、无糖酸奶1杯。
- **睡前加餐 (21:00)**：热牛奶1杯（如有需要）。

## 5. 特殊疾病调整

- **高血压**：减少咸菜、腐乳；汤中不放盐；多吃芹菜、香蕉。
- **糖尿病**：粥煮得不要太烂；主食增加粗粮比例；水果选低GI（如樱桃、李子）。
- **痛风**：肉类焯水弃汤；不喝浓肉汤；避免动物内脏、贝类。
- **肾病**：限制植物蛋白（豆类），优选动物蛋白（鱼、蛋清）；蔬菜焯水去钾。
