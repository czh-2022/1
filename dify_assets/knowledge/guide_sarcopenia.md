# 肌少症与老年人肌肉健康饮食指南 (Sarcopenia)

## 1. 疾病概述
肌少症（Sarcopenia）是指与年龄相关的骨骼肌质量减少、肌肉力量下降和躯体功能减退。这会导致老年人跌倒、骨折、残疾甚至死亡风险增加。“存钱不如存肌肉”，营养干预是防治肌少症的关键。

## 2. 核心饮食原则
- **足量优质蛋白**：每日蛋白质推荐摄入量为1.2-1.5g/kg体重（高于一般老年人的1.0g/kg）。
- **蛋白质分布**：建议将蛋白质均匀分配到三餐（每餐至少20-30g），而不是集中在晚餐，更有利于肌肉合成。
- **亮氨酸（Leucine）**：一种必需氨基酸，是启动肌肉合成的“开关”。富含亮氨酸的食物包括乳清蛋白、牛肉、鱼类。
- **维生素D**：维持肌肉功能，缺乏可能导致肌无力。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（建议多吃）
- **乳清蛋白**：牛奶、酸奶、乳清蛋白粉（吸收最快，亮氨酸含量高）。
- **瘦肉**：牛肉、羊肉、猪里脊（富含肌酸和铁）。
- **鱼虾**：三文鱼、鳕鱼、虾仁（高蛋白低脂肪）。
- **蛋类**：鸡蛋（生物价最高，极易吸收）。
- **大豆制品**：豆腐、豆浆（植物蛋白中质量最好的）。

### ❌ 红灯食物（限制/避免）
- **高糖零食**：饼干、蛋糕（导致胰岛素抵抗，抑制肌肉合成）。
- **劣质蛋白**：胶原蛋白（如猪蹄、鸡爪，缺乏必需氨基酸，不能补肌肉）。
- **油炸肉类**：炸鸡、炸猪排（高脂低质）。

## 4. 推荐食谱示例 (高蛋白餐单)
- **早餐**：牛奶冲麦片 + 煮鸡蛋1个。
- **午餐**：清蒸鲈鱼（100g） + 香干炒芹菜 + 米饭。
- **加餐**：希腊酸奶（100g）或蛋白粉一杯。
- **晚餐**：番茄炖牛腩（瘦肉50g） + 豆腐汤 + 馒头。

## 5. 特别提示
- **抗阻运动**：单纯吃不够，必须配合抗阻力训练（如弹力带、举哑铃、深蹲），才能有效刺激肌肉生长。
- **肾病注意**：如有慢性肾病，需在医生指导下调整蛋白质摄入量，避免加重肾脏负担。
