# 骨质疏松症与骨骼健康饮食指南

## 1. 疾病概述
骨质疏松症（Osteoporosis）是一种以骨量低、骨组织微结构损坏为特征，导致骨脆性增加、易发生骨折的全身性代谢性骨病。老年人是高发人群，尤其是绝经后女性。

## 2. 核心饮食原则
- **足量钙摄入**：钙是骨骼的基础。50岁以上人群每日推荐摄入量为1000-1200mg。
- **补充维生素D**：促进钙吸收。建议每日户外活动晒太阳20分钟，或遵医嘱补充维生素D制剂。
- **优质蛋白质**：维持骨基质合成，但需避免过量（过多蛋白可能增加尿钙流失）。
- **低盐饮食**：高钠饮食会促进钙排泄，建议每日食盐<5g。
- **限制咖啡因与酒精**：过量咖啡和酒精会影响钙吸收。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（建议多吃）
- **乳制品**：牛奶、酸奶、奶酪（钙的最佳来源）。
- **豆制品**：豆腐干、卤水豆腐（富含钙和异黄酮）。
- **绿叶蔬菜**：油菜、芥蓝、西兰花（富含维生素K和钙）。
- **坚果**：杏仁、芝麻酱（钙含量高）。
- **海产品**：带骨小鱼、虾皮（需注意盐分）。

### ❌ 红灯食物（限制/避免）
- **高盐食品**：咸菜、腌肉、加工零食。
- **碳酸饮料**：磷酸可能影响钙代谢。
- **浓茶与浓咖啡**：影响钙吸收。
- **草酸含量高的蔬菜**：菠菜、苋菜（建议焯水后食用，去除草酸）。

## 4. 推荐食谱示例
- **早餐**：高钙牛奶（250ml） + 全麦面包 + 煮鸡蛋。
- **午餐**：香干炒肉丝 + 蒜蓉西兰花 + 杂粮饭。
- **加餐**：原味酸奶（100g） + 一小把杏仁。
- **晚餐**：虾皮冬瓜汤 + 清蒸鱼 + 馒头。

## 5. 特别提示
- 如果乳糖不耐受，可选择无乳糖舒化奶或酸奶。
- 骨头汤补钙效果极差（主要是脂肪），不推荐作为补钙主要来源。
