# 老年人便秘与肠道健康饮食指南

## 1. 疾病概述
便秘（Constipation）是老年人最常见的消化系统问题，主要原因包括胃肠蠕动减慢、膳食纤维摄入不足、饮水不足、运动量减少、药物副作用等。长期便秘会增加肠癌、痔疮、心脑血管疾病的风险。

## 2. 核心饮食原则 (High-Fiber Diet)
- **足量膳食纤维**：每日摄入25-35g。
  - **不溶性纤维**（增加粪便体积，促进排便）：全谷物、蔬菜、坚果。
  - **可溶性纤维**（吸水软化粪便，保护肠粘膜）：燕麦、豆类、水果。
- **充足水分**：每日至少1.5-2L水（除非有心肾功能不全），纤维吸水后才能发挥作用，否则反而加重便秘。
- **适量益生菌/益生元**：改善肠道菌群，促进蠕动。
- **规律进食**：早餐后容易产生胃结肠反射，是排便的最佳时机。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（建议多吃）
- **全谷物**：糙米、燕麦、全麦面包、玉米、红薯（带皮）、杂豆。
- **高纤蔬菜**：芹菜、韭菜、菠菜、空心菜、菌菇类（木耳、香菇）。
- **通便水果**：火龙果（尤其是红心）、猕猴桃、梨、带皮苹果、西梅（天然泻药）。
- **坚果/种子**：芝麻、亚麻籽、奇亚籽（富含油脂，润肠通便）。
- **酸奶/发酵食品**：含有活菌的酸奶、纳豆。

### ❌ 红灯食物（限制/避免）
- **精制米面**：白米饭、白馒头、面条（缺乏纤维）。
- **加工肉类**：香肠、午餐肉（通常低纤高脂，延缓排空）。
- **未熟透的香蕉**：含鞣酸，具有收敛作用，反而加重便秘。
- **柿子、石榴**：含鞣酸，易形成粪石。
- **大量浓茶**：茶多酚具有收敛性。

## 4. 推荐食谱示例
- **早餐**：红薯燕麦粥 + 煮鸡蛋 + 一杯温开水（早起空腹喝）。
- **午餐**：西芹百合炒虾仁 + 凉拌黑木耳 + 杂粮饭（糙米+白米）。
- **加餐**：红心火龙果半个 + 酸奶一杯。
- **晚餐**：菠菜猪肝汤 + 炒杂蔬（玉米粒、胡萝卜、豌豆） + 玉米面窝头。

## 5. 特别提示
- **循序渐进**：纤维摄入量应逐渐增加，避免突然大量摄入导致腹胀、产气。
- **适当运动**：如散步、揉腹（顺时针），促进肠蠕动。
- **如厕习惯**：养成定时排便的习惯，即便没有便意。
- **药物排查**：某些药物（如钙通道阻滞剂、阿片类止痛药、抗抑郁药）可能引起便秘，需咨询医生。
