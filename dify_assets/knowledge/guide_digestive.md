# 消化系统健康与胃病饮食指南 (Digestive Health)

## 1. 疾病概述
老年人常患有慢性胃炎、胃溃疡、反流性食管炎（GERD）或消化不良。随着年龄增长，胃酸分泌减少，胃粘膜变薄，消化能力减弱。不当饮食会引发胃痛、反酸、腹胀。

## 2. 核心饮食原则
- **细嚼慢咽**：口腔是消化的第一站，减轻胃部负担。
- **定时定量**：规律进食中和胃酸，保护胃粘膜。
- **温和饮食**：避免过冷、过热、过辣、过硬。
- **干稀分食**（针对反流性食管炎）：吃饭时不喝汤水，饭后半小时再喝，防止胃容积过大引起反流。
- **易消化**：选择纤维适中、质地柔软的食物。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（养胃推荐）
- **发酵面食**：馒头、花卷、发糕（碱性，中和胃酸，易消化）。
- **软烂蔬菜**：南瓜、胡萝卜、土豆、冬瓜、去皮西红柿。
- **优质蛋白**：蒸蛋羹、煮鸡蛋、鱼肉、鸡肉丸子、豆腐。
- **养胃粥**：小米粥（护胃粘膜）、山药粥。

### ❌ 红灯食物（伤胃禁忌）
- **强刺激物**：辣椒、咖喱、生大蒜、生洋葱、芥末。
- **高酸食物**：柠檬、浓醋、山楂（胃酸过多者禁用）。
- **难消化食物**：糯米（粽子、汤圆）、油炸食品（炸糕、油条）、肥肉。
- **产气食物**：豆类、红薯（如易腹胀者少吃）。
- **烟酒浓茶**：直接损伤胃粘膜。

## 4. 推荐食谱示例
- **早餐**：发面馒头 + 小米南瓜粥 + 煮鸡蛋。
- **午餐**：清蒸鲈鱼 + 上汤娃娃菜（煮软） + 软米饭。
- **加餐**：苏打饼干（中和胃酸）或 蒸苹果。
- **晚餐**：肉末豆腐羹 + 西兰花碎 + 花卷。

## 5. 特别提示
- **饭后防反流**：反流性食管炎患者饭后不要立即躺下，至少直立坐或散步30分钟。
- **睡前禁食**：睡前3小时不进食，防止夜间反酸。
