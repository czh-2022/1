# 慢病专病饮食指南：慢性肾病 (CKD)

## 1. 核心原则 (低蛋白、低盐、低磷、低钾)

### **CKD 分期饮食策略**
1. **CKD 1-2期** (eGFR ≥ 60)
   - 正常蛋白饮食 (0.8g/kg/d)。
   - 控制血压、血糖。
   - 避免肾毒性药物。

2. **CKD 3-5期 (非透析)** (eGFR < 60)
   - 低蛋白饮食 (0.6g/kg/d)。
   - 补充α-酮酸（遵医嘱）。
   - 限制磷、钾摄入。
   - 控制盐摄入 (<5g/d)。

3. **透析期** (血液透析/腹膜透析)
   - 高蛋白饮食 (1.0-1.2g/kg/d)。
   - 补充水溶性维生素。
   - 严格控制水分、钠、钾、磷。

## 2. 营养素控制详解

### **低蛋白 (Low Protein)**
- **优质蛋白占比**：50%以上（鸡蛋、牛奶、瘦肉、鱼）。
- **非优质蛋白**：米面、豆类（含磷较高）。
- **替代主食**：麦淀粉、粉丝、粉条、藕粉（几乎不含植物蛋白，减轻肾脏负担）。

### **低磷 (Low Phosphorus)**
- **高磷食物**：坚果（核桃、杏仁）、动物内脏、加工肉类（火腿）、全谷物（糙米、燕麦）、巧克力、可乐、奶酪。
- **降磷技巧**：
  - 肉类焯水煮沸后弃汤食用。
  - 避免食用含磷添加剂的加工食品。
  - 适量食用蛋清（高蛋白低磷），少吃蛋黄（高磷）。

### **低钾 (Low Potassium)**
- **高钾食物**：
  - **水果**：香蕉、橘子、橙子、猕猴桃、哈密瓜、樱桃、红枣。
  - **蔬菜**：菠菜、苋菜、土豆、山药、竹笋、蘑菇、紫菜、海带。
  - **其他**：低钠盐（含氯化钾）、咖啡、茶、果汁。
- **降钾技巧**：
  - 蔬菜切碎浸泡30分钟，焯水后炒食。
  - 避免生食蔬菜沙拉。
  - 避免食用果皮。

### **低盐 (Low Sodium)**
- 每日食盐 < 5g（透析患者 < 3g）。
- 避免腌制食品、酱油、味精。

## 3. 水分管理 (透析患者)
- **每日饮水量** = 前一日尿量 + 500ml。
- **监测体重**：透析间期体重增长不超过干体重的3-5%。
- **缓解口渴**：含冰块、嚼口香糖、喷雾湿润口腔。

## 4. 常见误区
1. **完全不吃肉**：导致营养不良，反而加重病情。应按量摄入优质蛋白。
2. **盲目喝汤**：老火汤含大量嘌呤、磷、钾，坚决避免。
3. **豆制品不能吃**：现代观点认为大豆蛋白属于优质蛋白，但需注意磷含量（豆腐优于干豆）。
