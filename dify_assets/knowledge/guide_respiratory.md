# 呼吸系统健康与COPD饮食指南 (Respiratory Health)

## 1. 疾病概述
慢性阻塞性肺疾病（COPD）、慢性支气管炎、哮喘等呼吸系统疾病在老年人中非常普遍。呼吸肌工作负荷增加会导致能量消耗巨大，且呼吸困难常影响进食，极易导致**营养不良**和**肌肉萎缩**，进一步恶化肺功能。

## 2. 核心饮食原则
- **高脂低碳**（针对COPD）：碳水化合物代谢产生的二氧化碳（CO2）最多，脂肪最少。对于二氧化碳潴留（排不出去）的患者，适当增加脂肪比例（35-40%），减少碳水化合物，有助于减轻肺部负担。
- **少食多餐**：饱餐会顶住横膈膜，妨碍呼吸。建议每日5-6餐，每餐7分饱。
- **优质蛋白**：维持呼吸肌（如膈肌）力量，防止呼吸衰竭。
- **充足水分**：稀释痰液，利于咳出。每日至少1.5-2L（除非有心衰）。
- **抗氧化剂**：维生素C、E、胡萝卜素，对抗肺部慢性炎症。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（建议多吃）
- **优质脂肪**：橄榄油、牛油果、坚果（核桃、杏仁）、深海鱼油（Omega-3具有抗炎作用）。
- **优质蛋白**：鸡蛋、牛奶、瘦肉、豆腐。
- **富含VC蔬果**：橙子、猕猴桃、西兰花、青椒（抗氧化，保护肺组织）。
- **全谷物**：燕麦、糙米（相比精米白面，血糖波动小，CO2产生适中）。

### ❌ 红灯食物（限制/避免）
- **产气食物**：豆类（黄豆、黑豆）、洋葱、萝卜、碳酸饮料、红薯。腹胀会顶住横膈膜，加重呼吸困难。
- **高糖食品**：蛋糕、甜饮料。代谢产生大量CO2。
- **含盐过高**：钠会导致水钠潴留，加重肺水肿或支气管充血。
- **冷饮/刺激性食物**：可能诱发支气管痉挛（哮喘患者尤需注意）。

## 4. 推荐食谱示例 (高脂低碳)
- **早餐**：全麦面包一片 + 煎蛋一个 + 几片牛油果 + 一杯全脂牛奶。
- **加餐**：一小把核桃仁。
- **午餐**：香煎三文鱼 + 炒西兰花 + 少量杂粮饭。
- **加餐**：原味酸奶。
- **晚餐**：冬瓜炖排骨（吃肉喝汤） + 少量馒头。

## 5. 特别提示
- **餐前休息**：进食前休息30分钟，避免因呼吸急促而吃不下饭。
- **吞咽配合**：如有气喘，进食时需细嚼慢咽，避免呛咳。
