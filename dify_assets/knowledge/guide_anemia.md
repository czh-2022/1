# 老年人贫血与补铁饮食指南 (Anemia)

## 1. 疾病概述
老年人贫血（Anemia）主要表现为乏力、头晕、心慌、面色苍白，是老年人常见的临床症状之一。最常见的类型是缺铁性贫血（Nutritional Anemia），其次是巨幼细胞贫血（缺乏叶酸或维生素B12）。

## 2. 核心饮食原则 (High-Iron Diet)
- **增加血红素铁**：动物肝脏、红肉（瘦肉）、动物血（吸收率高，不受植酸影响）。
- **补充维生素C**：促进非血红素铁（植物性铁）吸收。餐后吃富含VC的水果（如猕猴桃、草莓）。
- **限制抑制因子**：鞣酸（茶、咖啡）、草酸（菠菜、苋菜）、植酸（全谷物）会抑制铁吸收。
- **针对巨幼贫**：补充叶酸（深绿蔬菜、豆类）和维生素B12（动物内脏、蛋黄）。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（建议多吃）
- **血制品**：猪血、鸭血（每周1-2次，铁含量极高且易吸收）。
- **红肉**：牛肉、羊肉、猪瘦肉（每日适量）。
- **动物肝脏**：猪肝、鸡肝（每月1-2次，胆固醇高需限量）。
- **高VC蔬果**：鲜枣、猕猴桃、柑橘、青椒、番茄。
- **黑木耳、芝麻酱**：植物性铁源，需配合VC食用。

### ❌ 红灯食物（限制/避免）
- **浓茶、咖啡**：餐前餐后1小时内避免饮用，鞣酸会沉淀铁。
- **高草酸蔬菜**：菠菜、苋菜（必须焯水去除草酸）。
- **牛奶**：钙会抑制铁吸收，补铁时避免同餐大量饮奶。

## 4. 推荐食谱示例
- **早餐**：全麦馒头 + 猪肝粥（少量） + 橙子一个。
- **午餐**：青椒炒肉丝（VC+铁） + 西红柿炒蛋 + 米饭。
- **加餐**：红枣几颗。
- **晚餐**：鸭血豆腐汤 + 凉拌黑木耳 + 馒头。

## 5. 特别提示
- **查明原因**：老年人贫血不仅仅是营养问题，可能是消化道肿瘤、溃疡出血的信号，务必先就医排查！
- **铁剂补充**：如食补无效，遵医嘱服用铁剂（可能引起黑便、便秘，属正常现象）。
