# 免疫力提升与术后康复饮食指南 (Immune Support)

## 1. 适用人群
适用于流感/肺炎康复期、手术后恢复期、肿瘤放化疗期间或体质虚弱、易感冒的老年人。核心目标是**支持免疫系统功能**和**促进组织修复**。

## 2. 核心饮食原则
- **高蛋白**：抗体（免疫球蛋白）的本质是蛋白质。每日摄入1.2-1.5g/kg体重。
- **复合维生素**：
  - **维生素C**：促进白细胞功能（柑橘、猕猴桃）。
  - **维生素A**：维持呼吸道粘膜完整性，抵御病毒（胡萝卜、肝脏）。
  - **维生素D**：调节免疫应答。
- **矿物质**：
  - **锌**：免疫细胞发育必需（贝类、瘦肉）。
  - **铁**：防止贫血导致的免疫力下降。
- **益生菌**：70%的免疫细胞在肠道，维护肠道菌群至关重要。

## 3. 推荐食物（红灯/绿灯）

### ✅ 绿灯食物（超级免疫食物）
- **优质蛋白**：鸡蛋、海参、虾、瘦牛肉、鸡胸肉。
- **菌菇类**：香菇、金针菇（富含多糖，增强免疫）。
- **深色蔬果**：西兰花、菠菜、胡萝卜、蓝莓。
- **辛香料**：姜、蒜、姜黄（具有天然抗炎、杀菌作用）。
- **发酵食品**：酸奶（无糖）、纳豆。

### ❌ 红灯食物（削弱免疫）
- **高糖甜食**：过量糖分会抑制白细胞吞噬细菌的能力。
- **深加工食品**：含反式脂肪酸、防腐剂，引起体内慢性炎症。
- **生冷刺身**：对于免疫力低下者，生食存在寄生虫和细菌感染风险，务必**煮熟煮透**。

## 4. 推荐食谱示例 (康复餐)
- **早餐**：杂粮粥 + 两个煮鸡蛋 + 凉拌黑木耳。
- **午餐**：黄芪炖鸡汤（去皮喝汤吃肉，黄芪补气） + 香菇炒油菜 + 米饭。
- **加餐**：猕猴桃一个 + 酸奶。
- **晚餐**：西红柿牛腩煲 + 蒸山药。

## 5. 特别提示
- **食补为主**：不要过度依赖蛋白粉或补充剂，天然食物吸收最好。
- **睡眠**：良好的睡眠是免疫力恢复的基础。
